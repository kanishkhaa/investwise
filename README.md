# Investwise

Investwise is a gamified investing literacy platform .It guides users through a structured learning journey, evaluates knowledge via a quiz, helps them visualize risk, and provides an offline practice simulator.

## What’s included

### 1) User onboarding (`/onboarding`)
A short guided questionnaire (text + multiple-choice steps) that collects:
- Name
- Investment goal
- Time horizon
- Risk comfort level
- Monthly capacity
- Experience level


### 3) Learning path (`/learn`)
A module-based curriculum 

**Key features:**
- Modules listed with lock/unlock behavior based on completion order.
- Each module opens in a modal dialog:
  - Shows lesson visual + title + explanation.
  - Progress bar for the current module.
  - “Next lesson” / “Complete module” CTA.
- Completing a module:
  - Marks module as done.
  - Adds XP (+50 per module).
  - Bumps streak.
  - Triggers confetti.
  - Awards badges based on how many modules were completed:
    - **Foundations** (after a few modules)
    - **Graduate** (after all modules)

**Progress UI:**
- Shows total modules complete, streak days, XP, readiness score, and earned badges.


### 4) Knowledge quiz (`/quiz`)
A multi-question knowledge test using `QUIZ` from `src/lib/curriculum.ts`.

**Key features:**
- One question at a time with selectable options.
- After picking an option, the UI reveals correctness and explanation.
- Finishing the quiz updates:
  - **readinessScore** (best score kept)
  - **quizBest** (best raw score)
  - adds XP: `score * 10`
  - bumps streak
- Badge + confetti when the user performs well:
  - **“Quiz Whiz”** awarded when the result is strong (≥ 80%)


### 5) Risk visualization (`/risk`)
Visual risk explainer for three asset categories:
- FD (Fixed Deposit)
- Index Fund
- Single Stock

**Key features:**
- Asset tabs switch the model parameters (mean + volatility).
- Sliders control:
  - amount invested
  - time horizon (years)
- Shows:
  - A rotating “Risk Meter” (low/medium/high)
  - Best / likely / worst outcome panels
  - A live-updating area chart (historical-style projection)


### 6) Compare outcomes (`/compare`)
Side-by-side projection for the same initial amount over a chosen period.

**Key features:**
- Sliders for amount and years.
- Line chart shows:
  - FD likely projection
  - Index and Stock likely + best + worst bands
- Summary cards display final values and % differences.


### 7) Trading simulator (`/simulate`)
A mock trading interface that simulates prices locally and lets users practice buy/sell decisions.

**Key features:**
- Select a mock instrument (tabs):
  - RELIANCE, TCS, INFY, HDFCBANK, NIFTYBEES
- Live chart:
  - Prices are generated by a local random process.
  - Chart updates every ~1.8s.
- Trade panel:
  - Enter quantity
  - Buy:
    - checks cash availability
    - updates positions and average buy price
    - awards badge **“Trader”**
  - Sell:
    - checks holdings
    - updates quantity (removes position if qty becomes 0)
    - shows success/warning toast depending on P&L
- Portfolio header:
  - total equity
  - cash
  - holdings value
  - total P&L vs starting cash
- Reset button restores the practice portfolio:
  - cash = Can be edited by the user
  - empty positions
